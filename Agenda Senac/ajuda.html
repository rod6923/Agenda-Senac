Perguntas Frequentes

Como fazer uma nova reserva do auditório?

Para fazer uma nova reserva:

  1. Acesse a página "Agenda" no menu lateral
  2. Clique no botão "+" no horário desejado
  3. Preencha os dados do evento
  4. Clique em "Solicitar Reserva"
  5. Aguarde a aprovação do gerente

Como aprovar ou rejeitar uma solicitação?

Como gerente, você pode:

  1. Acesse a página "Agenda"
  2. Visualize as solicitações pendentes (marcadas em laranja)
  3. Clique na solicitação para ver os detalhes
  4. Escolha "Aprovar" ou "Rejeitar"
  5. Adicione comentários se necessário

Como cancelar uma reserva já aprovada?

Para cancelar uma reserva:

  1. Acesse "Minhas Reservas" ou "Minhas Solicitações"
  2. Encontre a reserva que deseja cancelar
  3. Clique em "Cancelar"
  4. Confirme o cancelamento
  5. O sistema enviará notificação aos envolvidos

Como alterar meus dados pessoais?

Para alterar seus dados:

  1. Clique no seu nome no canto inferior esquerdo
  2. Selecione "Meu Perfil"
  3. Edite as informações desejadas
  4. Clique em "Salvar Alterações"

Como visualizar relatórios de uso?

Para acessar relatórios:

  1. Acesse a página "Relatórios"
  2. Selecione o período desejado
  3. Escolha o tipo de relatório
  4. Clique em "Gerar Relatório"
  5. Baixe o PDF ou visualize online
